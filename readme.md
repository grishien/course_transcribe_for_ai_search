# 🎬 VIDEO Transcribe for AI Search

Инструмент для автоматической транскрибации обучающих видео и поиска по ним с точностью до секунды.

Загружаешь JSON с транскриптами в AI-чат (Claude, ChatGPT и др.) — и задаёшь вопросы по содержанию курса. В ответ получаешь название файла и таймкод, где это разбирается.

---

## ⚙️ Требования

- Windows 10/11
- [Python 3.10+](https://www.python.org/downloads/) — при установке отметь галку **"Add Python to PATH"**
- [ffmpeg](https://github.com/BtbN/FFmpeg-Builds/releases) — скачай `ffmpeg-master-latest-win64-gpl.zip`, распакуй в `C:\ffmpeg\`

Структура ffmpeg должна быть такой:
```
C:\ffmpeg\
  bin\
    ffmpeg.exe
    ffprobe.exe
    ffplay.exe
```

---

## 🚀 Установка

1. Клонируй репозиторий или скачай ZIP
2. Запусти `install.bat` — один раз, создаёт виртуальное окружение и устанавливает зависимости
3. Положи видеофайлы в папку `videos\` (можно в подпапках — скрипт обходит рекурсивно)

---

## ▶️ Запуск

```
run.bat
```

Скрипт пройдёт по всем `.mp4 / .mkv / .mov` файлам в папке `videos\` и сформирует файл `transcripts.json`.

**Время работы:** зависит от объёма видео и мощности компьютера. Модель `medium` обрабатывает ~1 минуту видео за ~1-3 минуты на CPU. На GPU — значительно быстрее.

---

## 🔍 Как искать по транскриптам

После того как `transcripts.json` сформирован — загрузи его в чат с AI и используй этот промпт:

> Я прикладываю JSON-файл с транскриптами обучающих видео. Структура каждой записи:
> - `file` — относительный путь к видеофайлу
> - `start` / `end` — таймкод в секундах
> - `text` — текст сегмента
>
> Когда я задаю вопрос, найди все релевантные сегменты и ответь в формате:
>
> **Файл:** `[путь к файлу]`  
> **Таймкод:** `[MM:SS]`  
> **Контекст:** [1-2 предложения о чём говорится в этом месте]
>
> Сортируй результаты по релевантности. Если тема раскрывается в нескольких местах — покажи все.

## 🛠️ Настройки

В файле `transcribe.py` можно изменить модель Whisper:

| Модель | Качество | Скорость | Размер |
|--------|----------|----------|--------|
| `tiny` | низкое | очень быстро | 75 MB |
| `base` | среднее | быстро | 145 MB |
| `small` | хорошее | умеренно | 465 MB |
| `medium` | **высокое** ✅ | медленно | 1.5 GB |
| `large` | лучшее | очень медленно | 3 GB |

По умолчанию используется `medium` — оптимальный баланс для русского языка.

## ❓ Частые проблемы

**`ffmpeg` не найден** — убедись что путь `C:\ffmpeg\bin` указан правильно в `transcribe.py` (первая строка после импортов).

**Скрипт работает медленно** — это нормально на CPU. Если есть NVIDIA GPU, установи `torch` с поддержкой CUDA — ускорение в 5-10 раз.

**JSON пустой во время работы** — так и должно быть. Файл записывается только после обработки всех видео.
